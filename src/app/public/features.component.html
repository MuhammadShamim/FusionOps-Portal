FusionOps Features

  • Streamlined team collaboration
  • Real-time updates and notifications
  • Intuitive, modern user interface (Angular 20+, Bootstrap 5)
  • Kanban board with drag-and-drop, modal-based editing, and CSV export
  • Notes with search, multi-sort, column toggling, row selection, CSV export, and copy-to-clipboard
  • Storage management with advanced table features and CSV export
  • PagerDuty integration: secure token storage, events table with advanced features, CSV export, and copy-to-clipboard
  • Settings page with secrets, storage, and PagerDuty tabs
  • Secure data management and environment-based configuration
  • Customizable workflows and persistent user preferences
  • Sticky header/footer, responsive sidebar, and robust layouts
